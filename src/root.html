Welcome to the Cloud Audition Project API

This API provides a Messages Palindrome Checker Service. With it, you can:

  • Create a message
  • Retrieve a message
  • Update a message
  • Delete a message
  • List all messages

API Endpoints

  • GET /messages - List all messages
  • POST /messages - Create a new message
  • GET /messages/:id - Retrieve a specific message
  • PUT /messages/:id - Update a message
  • DELETE /messages/:id - Delete a message

For more details on how to use these endpoints, please check out the API Documentation.
